OBB

Oriented Bounding Box (OBB) is a type of bounding volume used in computer graphics and computational geometry. It is often used to simplify complex geometric objects by correlating them as a box much closer in size and orientation to the actual object. Unlike the Axis-Aligned Bounding Box (AABB), the OBB is not constrained to align with the axis, so the box can be rotated.
