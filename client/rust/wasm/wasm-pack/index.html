wasm-pack

wasm-pack is a command line tool aimed at assembling and packaging Rust crates that target WebAssembly. It helps to bridge the gap between WebAssembly (written in Rust) and JavaScript. It generates necessary files for publishing a package to npm. wasm-pack also helps to make sure your rust code is set up correctly to get compiled to wasm.
